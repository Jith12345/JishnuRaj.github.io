PROJECT 01

Hospital IT Risk Assessment

Full risk register for a 200-bed hospital with no dedicated security team. Identified 10 critical assets, scored all threats using Likelihood × Impact methodology, assigned treatment options, and wrote specific technical controls for each risk.

Key Findings
7 assets scored Critical (25/25). Highest priority: unpatched medical devices connected directly to the main hospital network without segmentation — creating a lateral movement path to patient records.

Windows Event Log Investigation

Key Findings

Ransomware Incident Response Simulation
